# 논문 템플릿 자동 입력 앱/프로그램 제안

## 목표
워드(독스) 논문 템플릿에 미리 작성한 논문 내용을 자동으로 입력하여, 템플릿 형식을 유지하면서 빠르게 문서를 생성한다.

## 추천 접근 방식
### 1) 템플릿에 "표식"(placeholder) 넣기
- 워드 템플릿(.docx)에 `{{title}}`, `{{abstract}}` 같은 표식을 삽입한다.
- 표식은 문서 텍스트, 표, 머리말/바닥글 등 필요한 위치에 배치한다.

### 2) 입력 데이터 구조화
- 미리 작성한 논문 내용을 `JSON` 또는 `YAML` 파일로 관리한다.
- 예시 구조:
  - `title`, `authors`, `affiliations`
  - `abstract`, `keywords`
  - `sections`(서론, 방법, 결과, 결론 등)
  - `references`

### 3) 자동 삽입 도구 선택
- **Python** 기반: `docxtpl`(Jinja2 스타일 템플릿) 또는 `python-docx`로 삽입.
- 장점: 로컬에서 빠르게 구현 가능, 배포도 간단.

## 구현 예시 흐름
1. 템플릿 문서(`template.docx`) 준비 및 표식 삽입.
2. 논문 내용 데이터를 `paper.json`으로 저장.
3. 스크립트 실행 → `output.docx` 생성.

## 확장 아이디어
- **GUI 앱**: Electron 또는 Streamlit으로 간단한 입력 화면 제공.
- **웹 앱**: FastAPI + 프론트엔드로 업로드/다운로드 자동화.
- **버전 관리**: 템플릿 버전별로 다른 표식 세트 관리.

## 다음 단계 제안

3. 자동 삽입 스크립트(Python) 프로토타입 작성.
